# 邮件模板

> 工作邮件模板
>
> 中文字体推荐使用**微软雅黑**。
>
> 英文字体推荐使用**Arial**、**Times New Roman** 。

## 英文模板

```
Hi, Dear XXX

... ...
Best Regards
=============================================
XXX
Position：XXXX
Department：XXXX
Phone：XXXX
Email：XXXX@XXXX
=============================================
```

```

。。。 。。。

　　敬礼！
=============================================

=============================================
```
